##Описание
Консольная утилита для фильтрации содержимого текстовых файлов.
Программа принимает один или несколько входных файлов, содержащих строки, целые числа и вещественные числа (разделённые переводом строки), также:
- записывает целые числа в отдельный файл
- записывает вещественные числа в отдельный файл
- записывает строки в отдельный файл
- выводит статистику по отфильтрованным данным

##Сборка проекта
В корне проекта выполнить:
```mvn clean package```
